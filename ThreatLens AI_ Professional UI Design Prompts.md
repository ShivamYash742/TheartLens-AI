# ThreatLens AI: Professional UI Design Prompts

To achieve a professional, enterprise-grade cybersecurity UI that avoids the generic "AI-generated" aesthetic, focus on a sophisticated visual language that emphasizes clarity, data integrity, and user control. The goal is a UI that feels powerful, trustworthy, and intuitive for security professionals.

---

## 1. System-Wide Design Prompt

**Prompt:**

> "Design a sophisticated, dark-themed user interface for a cybersecurity threat intelligence platform. The aesthetic should be clean, modern, and data-focused, reminiscent of high-end security operations centers or advanced analytics dashboards. Avoid overly futuristic or neon elements. Emphasize clear information hierarchy, subtle animations for data updates, and a sense of calm control amidst complex data. Use a limited, professional color palette with a single, high-contrast accent color for interactive elements and critical alerts. Ensure all elements are legible and functional, even in low-light environments. The overall impression should be one of precision, reliability, and advanced analytical capability."

---

## 2. Component-Specific Prompts

### A. Knowledge Graph Visualization

**Prompt:**

> "Create an interactive knowledge graph visualization for cybersecurity entities (vulnerabilities, malware, systems, attack vectors). The graph should be rendered against a dark background, with nodes represented by subtle, distinct shapes or icons (e.g., hexagonal for systems, circular for vulnerabilities, triangular for malware). Edges (relationships) should be thin, clear lines with directional arrows, subtly animated to indicate data flow or connection strength. Critical relationships or high-severity nodes should be highlighted with the accent color. Provide intuitive zoom, pan, and filter controls. The layout should be dynamic but stable, preventing excessive clutter. Focus on readability of labels and the ability to quickly discern complex interdependencies."

### B. Intelligent Security Dashboard

**Prompt:**

> "Design a modular, highly configurable security dashboard. Each widget (e.g., 'Top 5 Critical Vulnerabilities,' 'Recent Attack Trends,' 'Affected Systems Map') should have a clean, dark card-like background with clear titles and subtle borders. Use data visualization techniques (bar charts, line graphs, heatmaps) that are minimalist yet highly informative. Typography should be consistent and legible, with numerical data prominently displayed. Interactive elements (filters, drill-downs) should be clearly identifiable using the accent color. The dashboard should convey a sense of real-time awareness and actionable intelligence without overwhelming the user."

---

## 3. Color Palette & Typography Guide

### A. Professional Color Palette

*   **Primary Backgrounds**: `#1A1A2E` (Deep Navy), `#0F0F1A` (Charcoal Black)
*   **Secondary Backgrounds/Cards**: `#2C2C4A` (Dark Slate), `#3A3A5A` (Muted Purple-Grey)
*   **Text/Primary Elements**: `#E0E0E0` (Light Grey), `#B0B0B0` (Medium Grey)
*   **Accent/Action Color**: `#00FFFF` (Cyan), `#39FF14` (Electric Green), or `#FFD700` (Gold) – *Choose ONE for consistency.*
*   **Alert/Warning**: `#FF4500` (Orange-Red), `#FF0000` (Bright Red)

### B. Typography

*   **Primary Font**: `Inter`, `Roboto`, or `Montserrat` (Sans-serif, clean, highly legible)
*   **Headings**: Bold weights of the primary font.
*   **Body Text**: Regular weights of the primary font, with sufficient line height for readability.
*   **Monospace (for code/technical IDs)**: `Fira Code` or `JetBrains Mono`.

**Key Principle**: Use typography to establish hierarchy. Larger, bolder text for titles; smaller, lighter text for details. Maintain consistent font sizes across similar elements.

---

By adhering to these prompts and guidelines, your ThreatLens AI UI will project a professional, cutting-edge image that resonates with cybersecurity experts, moving far beyond a generic AI aesthetic.
